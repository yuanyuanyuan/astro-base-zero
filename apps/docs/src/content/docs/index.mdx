## 🎯 什么是 Astro Base Zero？

Astro Base Zero 是一个智能化的 Astro 项目快速发布平台，专为连续创业者和开发者设计。它解决了从创意到上线的完整工作流程，让你能够专注于创造价值，而不是重复的技术配置。

## 🚀 快速体验

```bash
# 1. 安装 CLI 工具
npm install -g @astro-base-zero/cli

# 2. 配置个人品牌信息
astro-zero config brand

# 3. 创建第一个项目
astro-zero create my-awesome-project --template base

# 4. 启动开发服务器
cd my-awesome-project
npm run dev
```
